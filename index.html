How it works

Small workshop, sharp tools

🛠️
Built at the table

Every feature starts as something I needed in my own game. If it doesn't survive a real session, it doesn't ship.

🔓
Open core

Core tools are free. Patreon unlocks content packs and early access to new work - the tools themselves stay open.

🧩
Built to work together

Modules share common foundations - consistent interface, fewer conflicts with other modules, less bloat on the install.
